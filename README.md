# Masquerade Emulator

A multi-system emulator which emulates the following platforms (as of now):
1) Chip8
2) Space Invaders
3) Pacman (Midway and Namco)
4) Ms Pacman
5) Gameboy
6) Gameboy Color
7) Game Of Life
